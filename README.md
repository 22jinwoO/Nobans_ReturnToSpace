# 🎮 Nobans_ReturnToSpace
포트폴리오 노반 : 리턴 투 스페이스 (2D 캐주얼 플랫포머 게임)

---

# 게임 소개
우주정거장에서 지구로 추락한 외계인 노반이 우주선으로 돌아가는 것을 목표로 하는 2D 캐주얼 플랫포머 게임

---

# 📌 구현 기능
- 오브젝트 풀링을 이용한 아이템 및 발판 랜덤 생성
- 배경 스크롤링
